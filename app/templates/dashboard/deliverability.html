Google Postmaster Tools — Setup Required

Connect Postmaster Tools to see domain reputation, Gmail spam rates, and SPF/DKIM/DMARC pass rates for your sending domains. This is ground-truth data directly from Google.

  1. Enable the Gmail Postmaster Tools API in your Google Cloud project
  2. Go to postmaster.google.com → add each sending domain and verify ownership
  3. In Postmaster Tools → Settings → Users → add your service account email as Viewer
  4. Set GOOGLE_POSTMASTER_DOMAINS in Railway: willowcreekinvest.com,willowcreekfunds.com,wcpinvestors.com
  5. GOOGLE_ANALYTICS_CREDENTIALS_JSON is already set — the same service account is reused
